## laravel-crud

Crud проект с моделью Position и Worker, связь OneToMany.

- Запуск проекта
- Главная страница
- Добавление данных в базу данных (seeder)

## Запуск проекта

- Переходим в корневую папку проекта:
  ~/PhpstormProjects/laravel-crud$
- Запускаем проект командой: php artisan serve
- Workers и positions добавлены

## Главная страница

- http://localhost:8000/workers

## Добавление данных в базу данных (seeder)

- Переходим в корневую папку проекта:
  ~/PhpstormProjects/laravel-crud$
- Вводим команду: php artisan db:seed
